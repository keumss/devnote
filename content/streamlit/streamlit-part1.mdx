## Streamlit의 역할과 이 섹션의 학습 범위 이해하기

데이터를 살펴보는 내부 도구, 분석 결과를 공유하는 대시보드처럼 Python 계산 결과를 빠르게 웹 화면으로 보여 줄 때 Streamlit을 사용합니다. HTML·JavaScript를 직접 작성하지 않아도 `st.`로 시작하는 함수 호출을 위에서 아래 순서대로 쓰면 입력 위젯과 결과 화면을 만들 수 있습니다.

이 섹션은 Python 3.12를 기준으로 화면 요소, 입력 위젯, 재실행, 상태, 캐시, 여러 화면, 배포 전 설정을 차례로 다룹니다. 데이터 변환 자체는 Pandas 등 일반 Python 코드가 담당하고, Streamlit은 그 입력·출력 경계를 화면으로 연결합니다.

## 가상환경에 Streamlit 설치하고 예제 실행하기

프로젝트별 가상환경을 만들면 다른 프로젝트의 패키지와 충돌하지 않습니다. 공식 예제 앱을 먼저 실행해 브라우저, Python, Streamlit이 연결되는지 확인합니다.

```bash
mkdir streamlit-study
cd streamlit-study
python3.12 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
python -m pip install streamlit
streamlit hello
```

브라우저에 예제 앱이 열리면 터미널에서 `Ctrl+C`를 눌러 서버를 종료합니다. Windows PowerShell에서는 가상환경 활성화 명령만 `.venv\Scripts\Activate.ps1`로 바꿉니다. 설치 버전과 서버 주소의 세부 값은 실행 환경에 따라 달라집니다.

## streamlit run으로 첫 화면 열기

`app.py`를 실행 파일로 정하고 `streamlit run`에 넘깁니다. 일반 Python 프로그램처럼 `python app.py`로 실행하면 브라우저 서버가 시작되지 않습니다.

```python
# app.py
import streamlit as st

st.set_page_config(page_title='독서 목표', page_icon='📚')
st.title('이번 주 독서 목표')
st.write('하루 30분씩 Python 책을 읽습니다.')
```

```bash
streamlit run app.py
```

브라우저에서 제목과 문장이 보입니다. 코드를 저장하면 개발 서버가 변경을 감지하며, 화면을 새로고침하면 최신 스크립트가 다시 실행됩니다.

`st.set_page_config()`는 앱의 탭 제목과 아이콘 같은 페이지 설정을 정하므로 한 파일에서 다른 Streamlit 명령보다 먼저 한 번만 호출합니다.

## st.number_input과 st.selectbox로 값을 입력받기

사용자의 선택으로 계산 결과를 바꿔야 할 때는 위젯의 반환값을 일반 변수처럼 사용합니다. 아래 예제는 권수와 난이도를 입력받아 이번 주 예상 학습 시간을 계산합니다.

```python
# app.py
import streamlit as st

st.title('학습 시간 추정기')
books = st.number_input('읽을 책 수', min_value=1, max_value=10, value=2)
level = st.selectbox('난이도', ['입문', '중급', '고급'])
minutes_per_book = {'입문': 90, '중급': 150, '고급': 240}[level]

st.write(f'예상 학습 시간: {books * minutes_per_book}분')
```

처음에는 `예상 학습 시간: 180분`이 보입니다. `number_input()`은 정수 범위 안의 숫자만, `selectbox()`는 제공한 선택지 중 하나만 반환하므로 계산 전에 입력 형식을 별도로 파싱할 필요가 없습니다.

## st.button으로 사용자가 요청한 작업만 실행하기

파일 저장이나 외부 API 호출처럼 매번 실행하면 안 되는 작업은 버튼의 반환값으로 감쌉니다. 버튼을 누른 실행에서만 `True`가 되며, 다음 상호작용에서는 다시 `False`가 됩니다.

```python
# app.py
import streamlit as st

st.title('간단한 환영 메시지')
name = st.text_input('이름', placeholder='예: 민지')

if st.button('메시지 만들기'):
    if name.strip():
        st.success(f'{name}님, 오늘의 학습 목표를 정해 보세요!')
    else:
        st.warning('이름을 입력해 주세요.')
```

빈 입력으로 버튼을 누르면 경고가, `민지`를 입력하고 누르면 성공 메시지가 보입니다. 다음 실행에도 결과를 남겨야 하는 경우에는 버튼의 일회성 값만으로 부족하므로 Part 2의 `st.session_state`에 값을 저장합니다.

## st.columns와 st.sidebar로 입력과 결과를 나누기

한 화면에 입력과 결과가 함께 있을 때는 사이드바에 전역 조건을 두고, 핵심 숫자는 열에 나란히 배치하면 빠르게 읽을 수 있습니다.

```python
# app.py
import streamlit as st

st.title('주간 학습 현황')
goal = st.sidebar.slider('목표 시간', min_value=1, max_value=20, value=8)
completed = st.sidebar.slider('완료 시간', min_value=0, max_value=20, value=5)

progress = min(completed / goal, 1.0)
left, right = st.columns(2)
left.metric('완료', f'{completed}시간')
right.metric('남은 시간', f'{max(goal - completed, 0)}시간')
st.progress(progress, text=f'달성률 {progress:.0%}')
```

기본값에서는 완료 5시간, 남은 시간 3시간, 달성률 62%가 표시됩니다. `st.sidebar`는 별도 컨테이너이고 `st.columns()` 안에는 화면 크기에 따라 좁아질 수 있는 핵심 정보만 둡니다.

## st.dataframe과 st.line_chart로 표와 추세 보기

표 데이터를 검토하고 날짜별 변화를 함께 보고 싶을 때는 `st.dataframe()`과 간단한 차트 요소를 조합합니다. 작은 예제는 Pandas 없이 딕셔너리도 바로 전달할 수 있습니다.

```python
# app.py
import streamlit as st

st.title('일별 집중 시간')
study_log = {
    '날짜': ['월', '화', '수', '목', '금'],
    '집중 시간(분)': [30, 45, 20, 60, 50],
}

st.dataframe(study_log, hide_index=True, use_container_width=True)
st.line_chart(study_log, x='날짜', y='집중 시간(분)')
```

표에는 다섯 날짜의 값이, 그 아래에는 상승·하락을 보여 주는 선 그래프가 나타납니다. 대화형 필터·정렬이 필요한 큰 데이터는 다음 단계에서 Pandas `DataFrame`으로 정리한 뒤 같은 방식으로 전달합니다.

## 화면 위에서 아래로 실행되는 규칙 확인하기

위젯 값을 바꾸거나 버튼을 누르면 Streamlit은 보통 스크립트를 위에서 아래까지 다시 실행합니다. 따라서 화면에 보이는 결과는 전역 변수에 이전 실행 결과를 쌓는 방식보다, 현재 위젯 값에서 다시 계산하는 순수한 Python 함수로 만드는 편이 안전합니다.

```python
# app.py
import streamlit as st

st.title('할인 가격 계산')
price = st.number_input('정가', min_value=0, value=10_000, step=1_000)
rate = st.slider('할인율', min_value=0, max_value=50, value=10)
sale_price = price * (100 - rate) // 100

st.write(f'할인 가격: {sale_price:,}원')
```

기본 입력의 할인 가격은 `9,000원`입니다. 재실행될 때마다 같은 입력으로 같은 결과가 계산되므로, 여기에는 상태를 저장할 필요가 없습니다. 사용자가 추가한 항목처럼 실행 사이에도 유지해야 할 값만 Part 2에서 상태로 관리합니다.

Streamlit 설치·실행과 화면 요소는 [Install Streamlit](https://docs.streamlit.io/get-started/installation), [API Reference](https://docs.streamlit.io/develop/api-reference)를 기준으로 확인합니다.
