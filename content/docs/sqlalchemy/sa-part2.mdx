## 1:N 관계설정 (One to Many)

ForeignKey와 relationship()을 사용해 테이블 간의 참조 관계를 파이썬 속성인 것처럼 쉽게 탐색할 수 있습니다.

```python
from sqlalchemy import ForeignKey
from sqlalchemy.orm import relationship

class Post(Base):
    __tablename__ = 'posts'

    id: Mapped[int] = mapped_column(primary_key=True)
    title: Mapped[str] = mapped_column(String(50))
    user_id: Mapped[int] = mapped_column(ForeignKey('users.id'))

    # User 객체에서 user.posts 로 접근 가능하게 양방향 매핑
    author: Mapped["User"] = relationship(back_populates="posts")

# User 클래스에 추가할 속성:
# posts: Mapped[list["Post"]] = relationship(back_populates="author")
```

## 데이터 속성 수정 (Update)

객체를 조회한 뒤, 속성 값을 변경하고 다시 세션에 커밋하여 반영합니다.

```python
with Session(engine) as session:
    # 1. 수정할 객체 조회
    stmt = select(User).where(User.name == 'Alice')
    alice = session.scalars(stmt).first()

    if alice:
        # 2. 파이썬 속성 변경
        alice.age = 26
        # 3. 변경사항 커밋
        session.commit()
```

## 데이터베이스 레코드 삭제 (Delete)

필요 없는 레코드를 세션의 delete() 메서드를 통해 삭제합니다.

```python
with Session(engine) as session:
    stmt = select(User).where(User.name == 'Charlie')
    charlie = session.scalars(stmt).first()

    if charlie:
        session.delete(charlie)
        session.commit()
```

## 대량 수정 (Bulk Update)

쿼리를 이용해 한 번의 명령으로 여러 레코드의 값을 동시에 갱신합니다. (성능 최적화)

```python
from sqlalchemy import update

with Session(engine) as session:
    # 30살 이상인 유저들의 나이를 모두 1살씩 증가시킴
    stmt = (
        update(User)
        .where(User.age >= 30)
        .values(age=User.age + 1)
    )
    session.execute(stmt)
    session.commit()
```
