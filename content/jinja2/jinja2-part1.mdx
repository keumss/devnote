## Jinja2의 역할과 렌더링 흐름

Jinja2는 서버가 전달한 데이터를 HTML로 바꾸는 템플릿 엔진입니다. 데이터 조회·권한 판단 같은 애플리케이션 로직은 Python에 두고, 템플릿은 표시와 반복적인 화면 구조에 집중합니다. FastAPI에서는 `Jinja2Templates`로 템플릿 폴더를 연결합니다.

```python
from fastapi import FastAPI, Request
from fastapi.templating import Jinja2Templates

app = FastAPI()
templates = Jinja2Templates(directory='app/templates')

@app.get('/')
def home(request: Request):
    return templates.TemplateResponse(
        request=request,
        name='index.html',
        context={'posts': []},
    )
```

## 템플릿 상속 (Template Inheritance)

블로그의 공통 레이아웃(헤더, 푸터, 사이드바 등)을 유지하기 위해 base.html을 만들고 다른 페이지들이 이를 상속(extends)받아 특정 블록(block)만 재정의합니다.

```html
<!-- app/templates/base.html -->
<!DOCTYPE html>
<html lang="ko">
<head>
    <title>{% block title %}My Blog{% endblock %}</title>
    <link rel="stylesheet" href="{{ url_for('static', path='style.css') }}">
</head>
<body>
    <header>
        <h1><a href="{{ url_for('render_home') }}">My Awesome Blog</a></h1>
    </header>

    <main>
        <!-- 자식 템플릿이 내용을 채울 영역 -->
        {% block content %}{% endblock %}
    </main>

    <footer>
        <p>&copy; 2026 My Blog</p>
    </footer>
</body>
</html>
```

## 변수 출력 및 필터 (Variables & Filters)

이중 중괄호 `{{ }}`를 사용하여 서버에서 전달된 변수(SQLModel 객체 등)를 출력합니다. 파이프(`|`)를 이용해 누락값을 처리하거나 데이터를 포맷팅할 수 있습니다. HTML 템플릿의 자동 이스케이프는 신뢰할 수 없는 입력을 그대로 출력하는 XSS를 막는 중요한 방어선이므로 해제하지 않습니다.

```html
<!-- app/templates/post_detail.html -->
{% extends "base.html" %}

{% block title %}{{ post.title }} - My Blog{% endblock %}

{% block content %}
    <article>
        <h2>{{ post.title }}</h2>
        <p class="meta">작성자: {{ post.author.username | default("익명") }} | 작성일: {{ post.created_at.strftime("%Y-%m-%d %H:%M") }}</p>

        <div class="content">
            <!-- 사용자 입력은 자동 이스케이프된 상태로 출력합니다. -->
            {{ post.content }}
        </div>
    </article>
{% endblock %}
```

## 자동 이스케이프와 안전한 HTML

`|safe`는 검증되지 않은 데이터에 사용하면 안 됩니다. 꼭 제한된 HTML 서식을 허용해야 한다면, 서버에서 허용 목록 기반으로 정제한 결과만 `Markup`으로 표시하고 원본 입력은 그대로 보관합니다.

```python
from markupsafe import Markup
import bleach

def sanitize_article_html(value: str) -> Markup:
    cleaned = bleach.clean(
        value,
        tags={'p', 'strong', 'em', 'a', 'code'},
        attributes={'a': ['href', 'title']},
        protocols={'http', 'https'},
        strip=True,
    )
    # 검증을 마친 값만 템플릿에 전달합니다.
    return Markup(cleaned)
```
