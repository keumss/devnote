## 반복문 (For Loops)

`{% for %}` 구문을 사용하여 데이터베이스에서 조회한 포스트 목록을 순회하며 렌더링합니다. 반복문 내에서 loop 객체를 사용하여 현재 인덱스 등을 확인할 수 있습니다.

```html
<!-- app/templates/index.html -->
{% extends "base.html" %}

{% block content %}
    <h2>최신 글 목록</h2>
    <ul class="post-list">
        {% for post in posts %}
            <li>
                <a href="/posts/{{ post.id }}">{{ post.title }}</a>
                {% if loop.first %}
                    <span class="badge">New!</span>
                {% endif %}
            </li>
        {% else %}
            <!-- posts 리스트가 비어있을 때 실행됩니다. -->
            <li>현재 등록된 글이 없습니다. 첫 글을 작성해보세요!</li>
        {% endfor %}
    </ul>
{% endblock %}
```

## 조건문 (If / Elif / Else)

`{% if %}` 구문을 사용하여 조건에 따라 다른 HTML을 렌더링합니다. 로그인 여부에 따라 글쓰기 버튼을 보이거나 숨길 때 유용합니다.

```html
{% if request.state.user %}
    <div class="user-menu">
        <p>환영합니다, {{ request.state.user.username }}님!</p>
        <a href="{{ url_for('create_post_form') }}" class="btn">새 글 작성</a>
        <a href="/logout" class="btn">로그아웃</a>
    </div>
{% else %}
    <div class="guest-menu">
        <a href="/login" class="btn">로그인</a>
        <a href="/signup" class="btn">회원가입</a>
    </div>
{% endif %}
```

## include로 공통 조각 조합하기

상속은 전체 레이아웃에, `include`는 알림·페이지네이션처럼 작은 화면 조각에 적합합니다. 포함되는 템플릿은 기본적으로 현재 컨텍스트를 받으므로, 필요한 변수 이름을 문서화해 결합도를 낮춥니다.

```html
<!-- app/templates/index.html -->
{% extends "base.html" %}

{% block content %}
    {% include "partials/pagination.html" %}
{% endblock %}

<!-- app/templates/partials/pagination.html -->
{% if page > 1 %}
    <a href="{{ url_for('render_home', page=page - 1) }}">이전</a>
{% endif %}
<span>{{ page }} / {{ total_pages }}</span>
{% if page < total_pages %}
    <a href="{{ url_for('render_home', page=page + 1) }}">다음</a>
{% endif %}
```

## 필터와 테스트로 표시 규칙 정리하기

간단한 표현식에는 내장 필터와 테스트를 사용하면 템플릿을 읽기 쉬워집니다. 복잡한 계산이나 DB 접근은 템플릿에 넣지 말고 라우터에서 값을 준비합니다.

```html
{% set display_name = user.nickname | default(user.username, true) %}

{% if posts is defined and posts | length > 0 %}
    <p>{{ display_name }}님의 글 {{ posts | length }}개</p>
{% else %}
    <p>아직 표시할 글이 없습니다.</p>
{% endif %}
```
