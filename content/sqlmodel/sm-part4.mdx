## Session 의존성 주입 (Depends)

FastAPI의 라우터에서 안전한 트랜잭션 관리를 위해 Session을 Generator(yield)로 주입받습니다.

```python
from fastapi import FastAPI, Depends

app = FastAPI()

def get_session():
    with Session(engine) as session:
        yield session

# 엔드포인트에서 db: Session 파라미터로 쉽게 주입받음
@app.get("/heroes/")
def read_heroes(session: Session = Depends(get_session)):
    heroes = session.exec(select(Hero)).all()
    return heroes
```

## POST 리소스 생성 엔드포인트 구현

검증된 데이터를 받아 DB에 저장하고, 새로 생성된 자원을 그대로 반환합니다.

```python
from fastapi import HTTPException

@app.post("/heroes/", response_model=Hero)
def create_hero(hero: Hero, session: Session = Depends(get_session)):
    # request body에서 파싱된 hero 객체를 그대로 Session에 추가
    session.add(hero)
    session.commit()
    # DB에 삽입되어 자동 생성된 ID값을 동기화
    session.refresh(hero)

    # JSON 응답으로 생성된 데이터를 반환
    return hero
```

## PATCH 요청에서 부분 수정과 404 처리

수정용 모델과 테이블 모델을 분리하면, 요청에 포함된 필드만 안전하게 바꿀 수 있습니다. 대상이 없으면 404를 반환하고, `exclude_unset=True`로 생략된 필드가 기존 값까지 덮어쓰지 않게 합니다.

```python
from typing import Optional
from fastapi import HTTPException

class HeroUpdate(SQLModel):
    name: Optional[str] = None
    secret_name: Optional[str] = None
    age: Optional[int] = None

@app.patch('/heroes/{hero_id}', response_model=Hero)
def update_hero(
    hero_id: int,
    hero_update: HeroUpdate,
    session: Session = Depends(get_session),
):
    db_hero = session.get(Hero, hero_id)
    if not db_hero:
        raise HTTPException(status_code=404, detail='Hero not found')

    db_hero.sqlmodel_update(hero_update.model_dump(exclude_unset=True))
    session.add(db_hero)
    session.commit()
    session.refresh(db_hero)
    return db_hero
```

## DELETE 요청의 성공 응답

삭제도 먼저 리소스 존재 여부를 확인한 뒤 한 트랜잭션으로 확정합니다. 클라이언트는 성공 여부만 필요하므로, 삭제된 객체 전체 대신 짧은 확인 응답을 돌려줄 수 있습니다.

```python
@app.delete('/heroes/{hero_id}')
def delete_hero(hero_id: int, session: Session = Depends(get_session)):
    hero = session.get(Hero, hero_id)
    if not hero:
        raise HTTPException(status_code=404, detail='Hero not found')

    session.delete(hero)
    session.commit()
    return {'ok': True}
```
