## Session 의존성 주입 (Depends)

FastAPI의 라우터에서 안전한 트랜잭션 관리를 위해 Session을 Generator(yield)로 주입받습니다.

```python
from fastapi import FastAPI, Depends

app = FastAPI()

def get_session():
    with Session(engine) as session:
        yield session

# 엔드포인트에서 db: Session 파라미터로 쉽게 주입받음
@app.get("/heroes/")
def read_heroes(session: Session = Depends(get_session)):
    heroes = session.exec(select(Hero)).all()
    return heroes
```

## POST 리소스 생성 엔드포인트 구현

검증된 데이터를 받아 DB에 저장하고, 새로 생성된 자원을 그대로 반환합니다.

```python
from fastapi import HTTPException

@app.post("/heroes/", response_model=Hero)
def create_hero(hero: Hero, session: Session = Depends(get_session)):
    # request body에서 파싱된 hero 객체를 그대로 Session에 추가
    session.add(hero)
    session.commit()
    # DB에 삽입되어 자동 생성된 ID값을 동기화
    session.refresh(hero)

    # JSON 응답으로 생성된 데이터를 반환
    return hero
```

## PATCH / DELETE 에러(404) 처리 핸들링

수정이나 삭제 요청이 들어왔을 때, 데이터베이스에 없으면 안전하게 404를 내려줍니다.

```python
@app.get("/heroes/{hero_id}", response_model=Hero)
def read_hero(hero_id: int, session: Session = Depends(get_session)):
    hero = session.get(Hero, hero_id)
    if not hero:
        # 객체를 못찾을 경우 FastAPI의 핵심 Error Raising 적용
        raise HTTPException(status_code=404, detail="Hero not found")

    return hero
```
